bridgedb.I18n¶

bridgedb.I18n.getLang(lang, localedir='/home/isis/share/locale')[source]¶

Return the Translation instance for a given language. If no Translation instance is found, return the one for ‘en’

bridgedb.I18n._(text)[source]¶

This is necessary because strings are translated when they’re imported. Otherwise this would make it impossible to switch languages more than once
